# Paukémon – Spielregeln

Ein kleines, nicht ganz ernst gemeintes Kartenspiel im Stil klassischer Monster-Duelle.

Zwei Spieler treten mit ihren Paukémon gegeneinander an. Jedes Paukémon hat Lebenspunkte, einen IQ-Wert, Fächer und eigene Attacken oder Reaktionen.

## Ziel des Spiels

Besiege alle Paukémon des Gegners.

Ein Paukémon gilt als besiegt, sobald seine LP auf **0** fallen.

## Spielaufbau

Jeder Spieler erhält ein Team aus mehreren Paukémon.

Zu Beginn ist jeweils ein Paukémon aktiv. Die übrigen Paukémon befinden sich im Team und können später eingewechselt werden.

Spieler 1 sitzt links, Spieler 2 rechts. In der Mitte befindet sich das Kampflog.

## Werte auf den Karten

### LP

LP sind die Lebenspunkte eines Paukémon.

Schaden wird von den aktuellen LP abgezogen. Heilung stellt LP wieder her, aber niemals über den Maximalwert hinaus.

### IQ

Der IQ ist ein Vergleichswert. Manche Attacken oder Reaktionen funktionieren nur, wenn ein Paukémon schlauer oder weniger schlau ist als ein anderes.

Der IQ verursacht nicht automatisch mehr Schaden, sondern wird nur verwendet, wenn eine Karte ausdrücklich darauf Bezug nimmt.

### Fächer

Die Fächer eines Paukémon können durch Ereigniskarten wichtig werden.

Zum Beispiel können bei einer Fachkonferenz alle Paukémon eines bestimmten Fachs betroffen sein.

## Ablauf einer Runde

Die Spieler sind abwechselnd am Zug.

In seinem Zug darf ein Spieler normalerweise genau **eine Aktion** ausführen:

* eine Attacke benutzen
* eine Fähigkeit benutzen
* ein anderes Paukémon einwechseln
* eine Ereigniskarte ziehen

Danach ist der andere Spieler an der Reihe.

## Attacken und Fähigkeiten

Jedes aktive Paukémon kann eine seiner aufgedruckten Attacken oder Fähigkeiten einsetzen.

Manche Attacken verursachen Schaden. Andere heilen, betäuben, verwirren oder haben besondere Effekte.

Wenn eine Attacke eine Bedingung hat, muss diese erfüllt sein. Ansonsten scheitert die Attacke oder hat keinen Effekt.

## Standardattacke

Die Standardattacke eines Paukémon ist immer die **erste aktive Attacke** auf der Karte.

Sternchen-Reaktionen zählen nicht als Standardattacke.

## Reaktionen mit Sternchen

Fähigkeiten mit einem `*` sind Reaktionen.

Sie kosten keine Aktion und können automatisch ausgelöst werden, wenn ihre Bedingung erfüllt ist.

Typische Reaktionen sind zum Beispiel:

* eine Attacke verhindern
* einer Attacke ausweichen
* Schaden zurückwerfen
* gegen bestimmte Gegner immun sein

Reaktionen werden geprüft, bevor der eigentliche Effekt einer Attacke eintritt.

## Aussetzen

Manche Effekte zwingen ein Paukémon dazu, eine oder mehrere Runden auszusetzen.

Ein Paukémon, das aussetzen muss, kann in seinem Zug keine Aktion ausführen.

Sternchen-Reaktionen bleiben aber weiterhin möglich.

## Aufladen

Einige Fähigkeiten müssen vorbereitet oder aufgeladen werden.

Während des Aufladens kann das Paukémon in der Regel nicht normal angreifen. Dafür wird die spätere Attacke stärker.

## Ereigniskarten

Ereigniskarten haben besondere Effekte auf das Spiel.

Sie können einzelne Paukémon oder sogar alle Paukémon betreffen.

Beispiele:

* Alle Paukémon verlieren LP.
* Bestimmte Fächer müssen aussetzen.
* Ein Paukémon darf in einer Runde eine zusätzliche Aktion ausführen.

Eine gezogene Ereigniskarte wird sofort ausgeführt.

## Wechseln

Ein Spieler kann sein aktives Paukémon gegen ein anderes lebendes Paukémon aus dem eigenen Team austauschen.

Das Wechseln verbraucht die Aktion des Spielers.

Besiegte oder entfernte Paukémon können nicht eingewechselt werden.

## Heilung

Heilung stellt LP wieder her.

Ein Paukémon kann durch Heilung nicht mehr LP erhalten, als auf seiner Karte angegeben sind.

## Besondere Effekte

Einige Karten haben ungewöhnliche Sonderregeln, zum Beispiel Überlaufen, vollständige Heilung oder das Entfernen von Paukémon aus dem Spiel.

Wenn ein solcher Effekt ausgelöst wird, gilt der Text der Karte.

## Sieg

Das Spiel endet, sobald ein Spieler keine lebenden Paukémon mehr übrig hat.

Der andere Spieler gewinnt.
